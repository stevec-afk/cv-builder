# CV Builder

A minimal, professional resume builder application that allows users to input their information and view a formatted preview in real time. This project was developed as part of The Odin Project curriculum.

## Project Details

The application features a split-panel design. The left panel houses form where you enter the data for personal details, professional experience, and education history. The right panel updates in real time to display a preview of a traditional resume.

The project page and complete curriculum guidelines can be found on [The Odin Project website](https://www.theodinproject.com/lessons/node-path-react-new-cv-application).

## Core Features

- Real-time text preview synchronisation across input forms and the document canvas.
- Encapsulated, state-driven collapsible form sections with explicit text triggers.
- Independent data entry tracks allowing users to add, edit, and delete multiple career or educational rows.
- Vite-optimized local SVG path asset pipeline to bypass third-party library footprints.
- Custom architectural desktop layout with fixed column proportions and a stationary scrolling gradient mask.
